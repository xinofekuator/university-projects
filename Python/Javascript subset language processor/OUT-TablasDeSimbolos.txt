El formato de las entradas de la tabla de simbolos es el siguiente: 
| Nombre | Tipo | Args/long | Return | 

Nombre : Nombre del identificador 
Tipo : Tipo del identificador 
Args/long: 
	 Id de tipo funcion: Tipo del argumento de la funcion 
	 Id de tipo Array: Longitud del array 
	 Otro caso: None 
Return: 
	 Id de tipo funcion: Tipo de lo que devuelve la funcion 
	 Otro caso: None
